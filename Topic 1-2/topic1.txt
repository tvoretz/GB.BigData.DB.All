2. Создайте базу данных example, разместите в ней таблицу users, состоящую из двух столбцов, числового id и строкового name.

Сейчас у вас может быть засчитано только второе задание первой темы.
По нему замечание:
id int(11) DEFAULT NULL,
name varchar(255) DEFAULT NULL
Подумайте, имеет-ли смысл выставлять поле, специально созданное для уникальной идентификации строк, в NULL?

Ответ: конечно не имеет смысла :) лучше сделать 
mysql> alter table users add primary key(id);
mysql> alter table users modify column `id` INT NOT NULL AUTO_INCREMENT;
